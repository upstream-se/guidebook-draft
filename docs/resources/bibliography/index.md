# Bibliography

> This page lists bibliographic references cited throughout the guidebook.

References cited across the guidebook are consolidated on this page.  Entries are listed in alphabetical order by the surname of the first author and formatted according to APA 7 ([American Psychological Association, 2020](#american-psychological-association-2020)).  Each entry includes sufficient publication metadata to enable identification and independent verification.
